# Arduino LM35 Temperature Monitor

Hệ thống giám sát nhiệt độ thời gian thực sử dụng cảm biến LM35 và giao diện máy tính.

## 📝 Mô tả dự án
Dự án thực hiện việc thu thập dữ liệu nhiệt độ từ cảm biến Analog LM35 thông qua board mạch Arduino. Dữ liệu sau đó được truyền lên máy tính qua giao tiếp Serial (UART) và hiển thị trực quan trên ứng dụng C# WinForms dưới dạng biểu đồ và giá trị số.

## 🛠 Thành phần hệ thống
* **Phần cứng:** - Arduino Uno / Nano.
  - Cảm biến nhiệt độ LM35.
* **Phần mềm:**
  - Firmware: Arduino IDE (C++).
  - Giao diện PC: Visual Studio (C# WinForms).
  - Mô phỏng: Proteus.

## 🚀 Tính năng
- Đọc nhiệt độ chính xác với sai số thấp.
- Hiển thị biểu đồ nhiệt độ thời gian thực trên máy tính.
- Giao diện người dùng thân thiện, dễ sử dụng.
